# linux目录结构

/ 根目录  即所有目录的顶点；

/bin   存放必要的命令；

/boot   存放内核以及启动所需的文件；

/dev   存放硬件设备文件；

/etc   存放系统配置文件；

- hosts：static information about host names //主机名与IP绑定信息 （文件）
- issue ：pre-login message and identification file //预登录消息和标识文件
- motd：post-login message of the day file(optional) 
- profile：systemwide initialization file for sh shell logins(optional)//用于sh shell登录的系统范围初始化文件
- resolv.conf ：resolver configuration file (optional) //dns域名服务器绑定

/home     普通用户的宿主目录，用户数据存放在其主目录中；

/lib|lib64     存放必要的运行库；

/mnt     存放临时的映射文件系统，通常用来挂载使用；

/proc     存放存储进程和系统信息；

/root     超级用户的主目录；

/sbin     存放系统管理程序；

/tmp     存放临时文件；

/usr     存放应用程序，命令程序文件、程序库、手册和其它文档； `不是user！`

/var     系统默认日志存放目录。

# ls 深入学习

# 1.命令可实现的功能

ls [选项] [目录名或文件名] 
#结合添加选项的不同进行不同功能的查找

-a  表示需要列出全部文件
-d  表示列出目录本身，而不是列出目录内的文件和子目录 #查看该目录自身的详细信息 
-l  列出详细信息
-h  把文件大小自动显示为以GB、MB、KB为单位

-S  （大写S）以文件大小排序
-t  按照文件修改时间排序
-i	回显inode号
ls -R：递归显示 #这个没见过

# 2.命令回显解释

-rwxrwxrwx. 1 wdd  wdd     1 4月   6 19:27 1.c

rwx 就是读、写、执行三个权限 顺序为文件所有者、文件所有者同组用户和其他用户

rwx之前的东东：

● 当为[d]则是目录
● 当为[-]则是文件；
● 若是[l]则表示为链接文档(link file)；
● 若是[b]则表示为装置文件里面的可供储存的接口设备(可随机存取装置)；
● 若是[c]则表示为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)。

一般就是-表示具体文件，d表示目录信息，这两个较为常用。

权限之后 就是所对应的软连接数量（链接数）

文件所有者名称 文件组名称 文件大小 文件创建时间 文件名称

# CP文件复制

cp [选项] 源文件目标文件

  -p  复制后的文件保留原文件的属性
  -d  如果原文件是链接文件，则只复制链接文件而不是复制它指向的实体文件
  -r  用来复制目录（复制目录的时候必须加上）
  -i  如果目标文件已经存在，询问是否覆盖
  -a  相当于pdr。这个参数常用

常用参数：

-r -a

针对目录的完全复制以及（保留原文件属性、只复制链接文件的链接性质、针对目录直接复制）

# rm删除

rm [选项] 文件或目录
 -f  rm删除文件或目录，默认会提示你是否确认删除，加上-f选项就不会提示了，rm会直接删除
 -r  删除目录时必须加上，用来递归删除目录下面的文件和子目录。生产中慎用！

# 查找find

find [路径] [参数] [被查找的对象]

# 可选参数：

-name：按名字查找	加*代表以[]*开头的所有文件
-iname 忽略大小写的按名字查找 
-inum inode查询
				-type f：只查找文件（常用）、
			 		  d：只查找目录（常用）
            		  s：查找socket文件
             		  c：查找字符设备文件

# 直接用

find / -name jason -exec ls -l {} \;

-exec 对查找到的结果进行处理

{}处理结果返回列表	\; exec的结尾分号进行转义

find /home -type f -mtime +7
近7天修改文件查询

-mtime n：查找修改时间在距离现在第4天
+n：查找4天前及更久以前被访问过的
-n：查找最近4天被访问过的

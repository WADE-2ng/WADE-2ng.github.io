# 管道符复习

命令 连接符解释
c1 & c2 1和2同时执行，==不论1的真假==，并返回所有结果
c1 && c2 先执行1，1为真再执行2，并返回所有结果,若1为假，返回1的错误信息，不执行2
c1 | c2 不管1的真假，只会执行2，返回2的信息，2错误无回显
c1 || c2 1，2都真，返回1，==1假，执行2==，返回1的错误信息
c1；c2 先1后2，1错误执行2，都会返回 ==Linux==

-eq 等于（equal）				-gt 大于（greater than）
-ge 大于等于（greater equal）	    -ne 不等于（Not equal）

# case语句

注意

1.匹配值右边必须加右括号

2.每次匹配之后的==;;==相当于break

3.==*)==相当于default

4.case结尾加==esac==

case $变量名 in 
"值1"） 
    如果变量的值等于值1，则执行程序1 
;; 
"值2"） # 整数可不加“” 
    如果变量的值等于值2，则执行程序2 
;; 
*） 
    如果变量的值都不是以上的值，则执行此程序 
;; 
esac

do

done
#列表循环

# 系统函数

# basename

获取绝对路径下文件名 （后缀可去）

basename /server/scripts/1.txt [无参数]
# 1.txt 返回路径末尾文件名/文件夹名 带后缀

basename /server/scripts/1.txt .txt
# 1 去后缀返回

basename /server/scripts/1qwe.txt we.txt
# 1q 最后一个 / 之后可以进行随意截断输出

# dirname

获取绝对路径的路径（最后一个 / 之前的内容进行输出）

dirname /server/scripts/1.txt
# /server/scripts
